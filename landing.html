What is a Cinema Managemnt System?

A Cinema Management System is where a user can subscribe to a topic and a someone else can publish information about that topic. In this case it is cinemas. So, a user could subscribe to a topic called "Cinema" and it would give them information about all the cinemas that have been published.

Who is this for?

This is not for customer use. This is for people inside the business to use and edit when necessary. It is an efficient way of updating/tracking topics within an organisation.

How is it useful?

This is more useful to cinema chain, but can also be used by smaller/ indepenednt cinemas if necessary. If a large cinema chain needs to update changes in a particular branch location, then a publisher can do so and everyone subscribed to that topic will be informed of it.
