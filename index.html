Overview of the system

To the right you can see a high level overview of what is going on in the backgorund. Right now you are using the front-end (website). As you can see a perosn can be a subscriber and/or publisher which updates the MQTT Broker. Basically MQTT is a lightweight, publish-subscribe network protocol that transports messages between devices. All that information gets sent to a data logger, which then sends it to our database, which you are able to access on this webpage.
